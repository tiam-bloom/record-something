## 说明

测试一个没有学过Android开发的开发的一个简单的AndroidAPP。

本项目使用 minimax 构建项目计划  
使用 opencode 完成项目开发  
使用ClaudeCode+minimax2.7解决环境以及运行问题  


**成本：**

- opencode 免费
- minimax Coding Plan 花费 26元/月，消耗 82次模型对话（限额：600次模型调用 / 5小时，6000次模型调用 / 周）
- 时间大约 3 个小时
